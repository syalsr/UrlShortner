# Сокращатель ссылок
Запуск
```shell
docker-compose up --build
```

# Использование
Для получения короткой ссылки нужно сделать get запрос
```shell
http://localhost:8080/short-url?URL=
```

Для использования короткой ссылки нужно сделать get запрос
```shell
localhost:8080/NHSGCdOjb2
```
